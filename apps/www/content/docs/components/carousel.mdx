## Introduction

The Carousel component is a versatile and customizable UI element for showcasing a series of images, content cards, or other items in a visually engaging, interactive slider format. Ideal for displaying featured content, product showcases, or galleries, the Carousel component supports smooth transitions and user controls for navigation, creating an intuitive experience. It includes options for autoplay, looping, and adjustable slide intervals, allowing developers to tailor its functionality and design to meet specific user engagement needs. This component is fully responsive, ensuring a seamless viewing experience across devices and screen sizes.

The carousel component is built using the [Embla Carousel](https://www.embla-carousel.com/) library.

## Usage

```tsx

  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,

```

## Examples

### Sizes

The CarouselSize component demonstrates a carousel with customizable dimensions, displaying a sequence of numbered cards arranged within a limited width. This carousel is responsive and aligns its items at the start, making it adaptable to various screen sizes. Each carousel item contains a card component with a centered, bold number to represent its position in the sequence. Navigation buttons are provided to allow users to cycle through the carousel content interactively. With a maximum width set for smaller screens, this example effectively showcases how carousel items can be dynamically rendered in a clean, grid-based layout.
To set the size of the items, you can use the `basis` utility class on the `<CarouselItem />`.

### Spacing

The CarouselSpacing component displays a carousel with spaced items, providing a clean, organized layout for numbered cards. Each item has a slight left margin, creating separation between cards while keeping them within a responsive, max-width container. Users can navigate through the carousel with next and previous buttons, making it easy to browse through content interactively.
To set the spacing between the items, we use a `pl-[VALUE]` utility on the `<CarouselItem />` and a negative `-ml-[VALUE]` on the `<CarouselContent />`.

<Callout className="mt-6">
  **Why:** I tried to use the `gap` property or a `grid` layout on the `
  <CarouselContent />` but it required a lot of math and mental effort to get the
  spacing right. I found `pl-[VALUE]` and `-ml-[VALUE]` utilities much easier to
  use.

You can always adjust this in your own project if you need to.

</Callout>

## API

The CarouselDApiDemo component demonstrates a carousel with an integrated API, allowing it to display the current slide position dynamically. As users navigate through the items, the component updates the current slide number and total count in real time. This setup provides a responsive carousel with numbered cards, next/previous buttons, and a footer showing the current slide status for an improved user experience.
Use a state and the `setApi` props to get an instance of the carousel API.

<ComponentPreview
  name="carousel-api"
  title="Carousel"
  description="A carousel with a slide counter."
/>

```tsx showLineNumbers {1,4,22}
import { type CarouselApi } from "@/components/ui/carousel"

export function Example() {
  const [api, setApi] = React.useState<CarouselApi>()
  const [current, setCurrent] = React.useState(0)
  const [count, setCount] = React.useState(0)

  React.useEffect(() => {
    if (!api) {
      return
    }

    setCount(api.scrollSnapList().length)
    setCurrent(api.selectedScrollSnap() + 1)

    api.on("select", () => {
      setCurrent(api.selectedScrollSnap() + 1)
    })
  }, [api])

  return (
    <Carousel setApi={setApi}>
      <CarouselContent>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
      </CarouselContent>
    </Carousel>
  )
}
```

## Events

You can listen to events using the api instance from `setApi`.

```tsx showLineNumbers {1,4-14,16}

export function Example() {
  const [api, setApi] = React.useState<CarouselApi>()

  React.useEffect(() => {
    if (!api) {
      return
    }

    api.on("select", () => {
      // Do something on select.
    })
  }, [api])

  return (
    <Carousel setApi={setApi}>
      <CarouselContent>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
      </CarouselContent>
    </Carousel>
  )
}
```

## Plugins

The CarouselPlugin component is a carousel with autoplay functionality, using a plugin to automatically cycle through items every 2 seconds. The autoplay pauses when users interact with the carousel (hovering stops it, and leaving resets it), allowing for both automated and manual navigation. With a compact layout and numbered cards, this component provides an engaging, hands-free browsing experience enhanced by next and previous buttons for easy interaction.
You can use the `plugins` prop to add plugins to the carousel.

```ts showLineNumbers {1,6-10}
import Autoplay from "embla-carousel-autoplay"

export function Example() {
  return (
    <Carousel
      plugins={[
        Autoplay({
          delay: 2000,
        }),
      ]}
    >
      // ...
    </Carousel>
  )
}
```
