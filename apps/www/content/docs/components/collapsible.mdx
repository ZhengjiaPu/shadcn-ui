## Introduction

The Collapsible component is an interactive UI element that allows users to toggle the visibility of content within a panel. By expanding or collapsing sections, this component helps manage screen space and improves content organization, especially useful for interfaces with large amounts of information or settings. The Collapsible component is ideal for FAQs, filters, or menus, providing a clean and efficient way for users to control the information they see. Customizable and easy to integrate, it enhances user experience by keeping interfaces clutter-free and intuitive.

## Usage

```tsx
import {
  Collapsible,
  CollapsibleContent,
  CollapsibleTrigger,
} from "@/components/ui/collapsible"
```

```tsx
<Collapsible>
  <CollapsibleTrigger>Can I use this in my project?</CollapsibleTrigger>
  <CollapsibleContent>
    Yes. Free to use for personal and commercial projects. No attribution
    required.
  </CollapsibleContent>
</Collapsible>
```
