## Introduction

The `Statistic` component is a dynamic display element designed for showcasing key values, statistics, or data points in a visually distinct format. It is particularly useful for dashboards, reports, or any data-rich interfaces. With several variants available, the Statistic component can be customized to include visual indicators, countdowns, and integration within card components, allowing for flexible data presentation tailored to various contexts. The component is also designed to follow accessibility guidelines, ensuring it can be used effectively by all users.

## Variants

### Statistic in Card

<ComponentPreview
  name="statistic-incard"
  title="Statistic in Card"
  description="A statistic component displayed within a card, using color-coded indicators."
/>
The `StatisticInCard` variant places the statistic component inside a card layout,
making it suitable for grouped data displays. The card includes color-coded indicators
to provide quick, visual insights into data trends or statuses. This setup is commonly
used in analytics panels or overview sections, where multiple statistics are displayed
side by side.

## Usage

```tsx
import {
  Statistic,
  StatisticCard,
  StatisticCountdown,
} from "@/components/ui/statistic"
```

```tsx
<Statistic value={12345} precision={2} />
<StatisticCard title="Sales" value={48000} indicator="positive" />
<StatisticCountdown targetDate="2024-12-31T23:59:59" />
```
