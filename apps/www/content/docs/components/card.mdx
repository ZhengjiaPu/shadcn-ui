## Introduction

The Card component serves as a flexible, modular container for displaying grouped content or information, providing a structured layout with a defined header, body, and footer. Designed to enhance readability and organization, the Card enables quick access to essential details, like notifications, messages, or brief updates, in a compact, visually appealing format. With customizable sections such as titles, descriptions, and actions, the Card component can be tailored to various use cases across interfaces, ensuring consistent design while maintaining flexibility for different content types.

## Usage

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from "@/components/ui/card"
```

```tsx
<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card Content</p>
  </CardContent>
  <CardFooter>
    <p>Card Footer</p>
  </CardFooter>
</Card>
```

## Variant-dynamic
The DynamicCardComponent is an interactive data card that displays event statistics with visual cues and dynamic footer content. Each event type is shown with key metrics, color-coded bar indicators, and percentage changes. On hover, additional details specific to each event type appear in the footer, providing context like latency or current status. This component is ideal for dashboards, offering an at-a-glance summary of critical metrics with responsive feedback to guide user focus.
