## Introduction

The HoverCardDemo component showcases a hover-triggered card, displaying additional information when users hover over the @nextjs button. The card features an avatar, a description, and an icon indicating the join date. This setup is perfect for giving users extra context without navigating away from the main content. It’s ideal for user profiles, tooltips, or any context where brief, on-hover information enhances the user experience.

## Usage

```tsx
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@/components/ui/hover-card"
```

```tsx
<HoverCard>
  <HoverCardTrigger>Hover</HoverCardTrigger>
  <HoverCardContent>
    The React Framework – created and maintained by @vercel.
  </HoverCardContent>
</HoverCard>
```
