## Introduction

The `Resizable` component is a flexible UI tool designed to create layouts with resizable panels. Built on top of [react-resizable-panels](https://github.com/bvaughn/react-resizable-panels), it provides a keyboard-accessible interface for creating responsive layouts. Developers can configure horizontal or vertical resizing, allowing for greater control and adaptability in web applications.

The component supports nested layouts and custom handle designs, making it an ideal choice for building interactive dashboards, editors, or any application requiring dynamic layouts. It also includes built-in accessibility features and customizable styles.

## Variants

### Vertical

Use the `direction` prop to set the direction of the resizable panels to vertical.

<ComponentPreview
  name="resizable-vertical"
  description="A group of resizable vertical panels."
/>

The vertical variant allows panels to resize along the vertical axis. It is commonly used for layouts with resizable headers and footers or splitting content into top and bottom sections.

```tsx showLineNumbers {9}
import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from "@/components/ui/resizable"

export default function Example() {
  return (
    <ResizablePanelGroup direction="vertical">
      <ResizablePanel>One</ResizablePanel>
      <ResizableHandle />
      <ResizablePanel>Two</ResizablePanel>
    </ResizablePanelGroup>
  )
}
```

### Handle

You can set or hide the handle by using the `withHandle` prop on the `ResizableHandle` component.

<ComponentPreview
  name="resizable-handle"
  description="A group of resizable panels with a custom handle."
/>

The handle variant introduces custom handles between panels, offering visual cues for resizing and improving usability. This variant is ideal for creating more interactive and user-friendly layouts.

```tsx showLineNumbers {11}
import {
  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from "@/components/ui/resizable"

export default function Example() {
  return (
    <ResizablePanelGroup direction="horizontal">
      <ResizablePanel>One</ResizablePanel>
      <ResizableHandle withHandle />
      <ResizablePanel>Two</ResizablePanel>
    </ResizablePanelGroup>
  )
}
```

## Usage

```tsx

  ResizableHandle,
  ResizablePanel,
  ResizablePanelGroup,
} from "@/components/ui/resizable"
```

```tsx

</ResizablePanelGroup>
```
