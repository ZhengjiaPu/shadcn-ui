## Introduction

The Radio Group component is a collection of radio buttons that allows users to select only one option from a predefined set. Each button represents a single choice, and selecting one automatically deselects any previously chosen option, ensuring only one option is active at a time. This component is especially useful for forms and settings where a single, definitive selection is required. Radio Groups provide a clear and intuitive way to present mutually exclusive options, enhancing usability by guiding users towards making a specific choice within a controlled selection set.

## Usage

```tsx
import { Label } from "@/components/ui/label"
import { RadioGroup, RadioGroupItem } from "@/components/ui/radio-group"
```

```tsx
<RadioGroup defaultValue="option-one">
  <div className="flex items-center space-x-2">
    <RadioGroupItem value="option-one" id="option-one" />
    <Label htmlFor="option-one">Option One</Label>
  </div>
  <div className="flex items-center space-x-2">
    <RadioGroupItem value="option-two" id="option-two" />
    <Label htmlFor="option-two">Option Two</Label>
  </div>
</RadioGroup>
```

## Variants

### Radio Group Form

The Radio Group Form variant is a customized version of the Radio Group component designed specifically for form use cases. It enables users to select a single option from a set of predefined choices within a form, making it ideal for survey questions, preference selection, or any scenario requiring mutually exclusive choices.
