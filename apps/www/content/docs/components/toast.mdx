## Introduction

The `Toast` component provides a concise message to notify users of an event or update. Toasts are temporary, appearing briefly at the edge of the screen before automatically disappearing. This component is ideal for confirmations, alerts, or notifications in applications where unobtrusive feedback is needed. The toast component follows accessibility standards to ensure inclusivity.

## Usage

The `useToast` hook provides a `toast` function that can be used to display a toast.

```tsx
import { useToast } from "@/components/ui/use-toast"
```

```tsx {2,7-10}
export const ToastDemo = () => {
  const { toast } = useToast()

  return (
    <Button
      onClick={() => {
        toast({
          title: "Scheduled: Catch up",
          description: "Friday, February 10, 2023 at 5:57 PM",
        })
      }}
    >
      Show Toast
    </Button>
  )
}
```

<Callout>

To display multiple toasts at the same time, you can update the `TOAST_LIMIT` in `use-toast.tsx`.

</Callout>
