## Introduction

The `Sonner` component is a customizable toast notification system for React, created by [emilkowalski\_](https://twitter.com/emilkowalski_). It provides an easy way to display temporary alerts and messages, enhancing the user experience by providing contextual feedback.

## Demo

The demo below provides an example of the `Sonner` toast notification component. This setup shows how toast notifications can be triggered to display important information or alerts.
