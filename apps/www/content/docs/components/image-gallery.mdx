## Introduction

The Image Gallery component offers a versatile solution for displaying a series of images in a structured, grid-like format. Designed to dynamically adjust according to screen size and orientation, the gallery offers users a responsive and interactive way to view collections. With built-in options for carousels, previews, tabs, and masonry grids, this component allows for seamless integration and customization in various applications, from simple galleries to complex interactive displays.

The component comes with styling that supports hover and focus states for enhanced accessibility and visual feedback during interactions. Whether showcasing images in a traditional grid or a visually engaging masonry layout, the Image Gallery provides a flexible, user-friendly experience suitable for various interface needs.

## Variants

### Gallery with Carousel

The carousel variant provides a sequential, scrollable view of images. With autoplay and manual controls, this layout is perfect for showcasing images in a focused, interactive manner.

<ComponentPreview
  name="image-gallery-with-carousel"
  description="An image gallery with carousel functionality."
/>

### Gallery with Preview

The preview variant displays a selected image in an enlarged view with thumbnails below for easy navigation. This layout is particularly useful for detailed image inspection and provides an intuitive way to navigate through a collection.

<ComponentPreview
  name="image-gallery-with-preview"
  description="An image gallery with a preview option for larger image views."
/>

### Gallery with Tabs

Organize images by category with the tabbed gallery variant. This structure allows users to switch between categories, offering a practical solution for thematic collections.

<ComponentPreview
  name="image-gallery-with-tab"
  description="An image gallery with categorized tabs for organized viewing."
/>

### Masonry Grid Gallery

The masonry grid variant arranges images in a staggered format, creating a fluid, visually appealing layout. This is an excellent choice for collections with images of varying dimensions, offering a modern, dynamic presentation.
